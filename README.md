# Analise-de-Modelos-de-Machine-Learning
Análise comparativa de modelos de Aprendizado de Máquina na predição de níveis de estresse com base em indicadores de sono, saúde e estilo de vida. Utilizando a base de dados pública Sleep Health and Lifestyle Dataset, avaliando o desempenho dos algoritmos K-Nearest Neighbors (KNN), Support Vector Machines (SVM) e Árvore de Decisão (Decision Tree).
